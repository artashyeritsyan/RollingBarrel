The Rolling Barrel is a small maze game I created where the main goal is simple: collect 50 beer mugs before time runs out.

You start inside a maze, and right away the timer is already going. At the top of the screen there’s a green bar showing how much time you have left. When that bar reaches zero, the game is over.

Around the maze there are beer mugs scattered everywhere. Every time you collect one, your barrel fills up a little. You can see it as a yellow bar at the top of the screen. Watching it grow feels rewarding, because it shows you’re getting closer to the goal.

But you’re not alone in the maze.

There’s a Skeleton walking around, trying to catch you. If it touches you, that’s it — you lose. So while you’re trying to collect mugs as fast as possible, you also have to avoid it.

You have a Dash ability. It helps you dodge the Skeleton when it gets too close or escape from danger. But it costs beer. Every time you use Dash, you lose some of the filled beer in your barrel, and the yellow bar goes down. So you need to think carefully before using it.

Reach 50 mugs before time runs out, and you win.
Get caught or run out of time, and you start again.
